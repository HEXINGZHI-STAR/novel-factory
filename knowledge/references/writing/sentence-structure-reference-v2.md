## 核心发现：AI写作 vs 人类写作 的句式偏差

这是最重要的一张表，直接决定你的文字像不像真人写的：

| 维度 | 真实网络文学范围 | AI常见偏差 | 修正方向 |
|------|----------------|-----------|---------|
| **平均句长** | 19-50字 | 8-18字（太短！） | **加长句子到25-40字** |
| **短句率(≤10字)** | 13%-37%，**平均20%** | 60%-70%（过高！） | 降低到20%以下 |
| **中句率(11-30字)** | 25%-49%，**平均38%** | 20%-30% | 提升到35%-45% |
| **长句率(30+字)** | 19%-61%，**平均42%** | 5%-15%（太低！） | **提升到40%-50%** |
| **说明/背景句占比** | 18%-34%，**平均26%** | 低（过度偏向描写） | 提升叙述推进句 |
| **描写句占比** | 18%-41%，**平均26%** | 过高（过度描写） | 降低纯描写 |
| **对话占比** | 3%-18%，**平均10%** | 偏高或偏低 | 根据题材调整 |
| **揭示/极短句** | 7%-18%，**平均11%** | 过度使用或完全不会 | 作为节奏点嵌入，不要滥用 |
| **问号占比** | 8%-26% | 过低 | 悬疑/推理类要提升到15%-25% |
| **叹号占比** | 0.3%-49%，**平均12%** | 过高或过低 | 军事/玄幻可适当使用，悬疑/规则怪谈克制 |

## 各题材详细参数表

选择你要写的题材，照着参数写就对了：

### 玄幻/仙侠类（《斗破苍穹》《斗罗大陆》《星辰变》《寸芒》《仙路烟尘》）
| 参数 | 范围 | 推荐目标 |
|------|------|---------|
| 平均句长 | 26-35字 | **30字** |
| 短句率 | 18%-26% | **20%** |
| 中句率 | 30%-46% | **40%** |
| 长句率 | 35%-50% | **40%** |
| 说明/背景占比 | 18%-28% | **25%** |
| 描写占比 | 21%-23% | **22%** |
| 对话占比 | 5%-18% | **12%** |
| 动作占比 | 14%-19% | **16%** |
| 句号 | 53%-87% | **70%** |
| 叹号 | 0.3%-27% | **仙侠类可以15%，玄幻克制到5%** |
| 问号 | 8%-12% | **10%** |

**核心写作模式**：
- 先交代背景/设定（说明句）→ 再描写场景 → 角色动作 → 简短对话 → 再推进
- 仙侠类（《寸芒》《仙路烟尘》）叹号可以高一些（20%-27%），更有感染力
- 玄幻类（《斗破苍穹》《斗罗大陆》）更克制，句号主导
- 长句是主力：30-50字的叙述推进句占比最高
- 极短句（2-10字）作为揭示点嵌入，10%-12%比例即可

**真实句式样例（来自《斗破苍穹》）**：
```
(说明/背景) 这是斗气大陆的常识。
(描写) 喧闹的人群中，测试员的声音，再次响了起来。
(动作+心理) "好不容易修炼而来的斗之气，又在消失…
(揭示) 墨落，笔停！
(叙述推进) 脸色一沉，一位长老怒喝道。
```

### 历史/权谋类（《庆余年》《大争之世》）
| 参数 | 范围 | 推荐目标 |
|------|------|---------|
| 平均句长 | 35-36字 | **35字** |
| 短句率 | 12%-16% | **15%** |
| 中句率 | 32%-33% | **32%** |
| 长句率 | 52%-54% | **53%** |
| 说明/背景占比 | 25%-32% | **28%** |
| 描写占比 | 28%-33% | **30%** |
| 对话占比 | 2%-6% | **4%** |
| 句号 | 67%-76% | **72%** |
| 叹号 | 3.5%-7.8% | **5%** |
| 问号 | 12%-17% | **15%** |

**核心写作模式**：
- **句子最长的题材**：历史/权谋需要铺垫和逻辑推进
- 对话极短（2%-6%）——角色说话少，更多是叙述者在讲故事
- **描写占比最高**（28%-33%）——环境、服饰、动作细节承载信息
- 问号12%-17%——角色在观察、推理、质疑
- 段落可以很长（庆余年649字/段），但也要有短段落制造变化

**真实句式样例（来自《庆余年》）**：
```
(长描写) 范闲赶紧咳了两声，阻止了老师的自吹自擂。
(叙述推进) 不过打也打了，就不需要考虑太多。
(长句+对话混合) 噢，他八岁的时候，他父亲请我去给他看过病…
(揭示短句) 真的很疼。
(揭示短句) 打错人了？
(长句推理) 黑骑士！
(短句) 神庙，就是供神的庙。
```

## 写作即时自检表（写完一章必查）

### 句式平衡自检
- [ ] 平均句长有没有达到**25字以上**？（统计100个句子的平均）
- [ ] 长句（30字+）有没有达到**40%以上**？
- [ ] 短句（10字以内）有没有控制在**20%以下**？
- [ ] 中句（11-30字）有没有达到**35%-45%**？

### 类型比例自检（按题材选择）
- [ ] 说明/背景推进句有没有占**20%-30%**？（科幻/历史要更高）
- [ ] 描写句有没有占**20%-30%**？（悬疑/奇幻要更高）
- [ ] 动作句有没有占**10%-20%**？（体育/爽文要更高）
- [ ] 对话有没有占**5%-15%**？（军事/科幻可以更低，奇幻更高）
- [ ] 揭示/极短句有没有占**7%-15%**？（悬疑/军事更高）

### 段落节奏自检
- [ ] 有没有连续3段以上都是50字以上的长段？→ 中间插一个极短段
- [ ] 有没有连续10句都是20字以内的短句？→ 加一些25-40字的推进句
- [ ] 有没有超过800字的超长段？→ 从逻辑转折处切开
- [ ] 段落长度的变化幅度够不够？→ 应该有30字短段，也有200-300字中长段

### 标点自检
- [ ] 问号在悬疑/推理类有没有达到**15%-25%**？
- [ ] 叹号在规则怪谈类有没有低于**5%**？
- [ ] 军事/仙侠/游戏类叹号可以在**10%-25%**，但不要通篇叹号

## 句式改写实战：从"AI味"到"人类味"

### 案例1：玄幻/仙侠开篇

**原文（AI写法 - 问题：句子太短，全是动作碎片）**：
> 少年睁开眼睛。他看了看四周。他发现自己躺在一张石床上。他坐了起来。他感到头很疼。一个老人走了进来。老人看着他。老人说："你醒了。"少年点了点头。他问："这是哪里？"老人说："青云宗。"

**诊断**：10句话，平均8字。短句率100%。描写0%。说明/背景0%。对话占比过高。

**改写（参考《斗破苍穹》句式比例：20%短句，40%中句，40%长句；25%说明，22%描写，16%动作，12%对话）**：
> 少年缓缓睁开眼，首先闻到了一股淡淡的药草气息，然后才发现自己躺在一张简陋的石床上，浑身骨头仿佛散了架一般。
> 
> 这是哪里？他想坐起来，却发现身体虚弱得惊人，稍一用力，脑袋就像被针扎一样疼。
> 
> 门外传来脚步声，一个穿着灰色长袍的老人走了进来，手里端着一只冒着热气的瓷碗。老人看到他醒了，脸上没有任何表情，只是淡淡地说了一句：
> 
> "你醒了。"
> 
> 少年点了点头，声音沙哑："请问…这是哪里？"
> 
> 老人将瓷碗放在床边的木桌上，目光在少年脸上停留了几秒，然后收回视线，声音不高不低：
> 
> "青云宗，后山。你被人从山下抬上来的时候，只剩一口气。"
> 
> 少年愣住了。
> 
> 青云宗？
> 
> 那个传说中的修仙门派？

**改写要点**：
- 平均句长从8字提升到25-30字
- 加入说明/背景（"那个传说中的修仙门派？"）
- 加入描写（药草气息、石床、灰色长袍、冒着热气的瓷碗）
- 对话从每句都有变成嵌入叙述中
- 用极短句（"少年愣住了。""青云宗？""那个传说中的修仙门派？"）做节奏点

### 案例2：悬疑/无限流开篇

**原文（AI写法 - 问题：句子太规整，全是陈述句）**：
> 房间里很暗。墙上挂着一个老式钟。钟在滴答地走。突然，钟停了。他抬起头。他看到窗外有一个影子。影子在动。他走过去。他拉开窗帘。什么都没有。他皱了皱眉。他转过身。他愣住了。房间中央站着一个人。

**诊断**：13句话，平均5字。短句率100%。问号0%。没有长句铺垫。

**改写（参考《无限恐怖》句式比例：18%短句，25%中句，57%长句；41%描写，26%问号，14%极短句）**：
> 空气里弥漫着一股灰尘和铁锈混合的味道，很旧，像是被遗忘了很多年。
> 
> 墙上挂着一只老式挂钟，铜制的钟摆左右晃动，发出规律而单调的"滴答、滴答"声，在寂静的房间里显得格外清晰。
> 
> 他不知道自己为什么会在这里。
> 
> 他只记得自己在电脑前打了个盹，然后…然后就什么都不记得了。
> 
> 他抬起头，看向窗外——窗外一片漆黑，什么都看不见，只有玻璃上映出的自己的脸，苍白得像纸。
> 
> 钟摆停了。
> 
> 他猛地回过头。
> 
> 挂钟的指针指着凌晨三点十七分，钟摆悬在半空，像是被什么东西夹住了。
> 
> 什么东西？
> 
> 他走到窗前，伸手拉开窗帘——
> 
> 窗外什么都没有。
> 
> 不对。
> 
> 他的瞳孔骤然收缩——窗外不是什么都没有，而是有什么东西，贴在玻璃上，距离他的脸不到十厘米。
> 
> 一张脸。
> 
> 一张没有眼睛的脸。

**改写要点**：
- 平均句长从5字提升到30-50字
- 加入大量长句描写（环境、感官细节）
- 用极短句（"钟摆停了。""一张脸。""一张没有眼睛的脸。"）制造冲击
- 问号提升到10%-15%（"什么东西？"）
- 段落长度从均匀短段变成长短交替：最长段50字，最短段3字

### 案例3：历史/权谋类开篇

**原文（AI写法 - 问题：对话过多，叙述不流畅）**：
> 他走进书房。书桌上放着一封信。他拿起信。他看了一眼。他的脸色变了。管家走了进来。管家说："老爷，有人找您。"他说："谁？"管家说："不知道，只说是宫里来的。"他沉默了。他说："让他进来。"

**诊断**：13句话，平均6字。短句率100%。描写0%。说明/背景0%。对话占比过高。

**改写（参考《庆余年》句式比例：15%短句，32%中句，53%长句；30%描写，28%说明，4%对话）**：
> 庆国十八年，春。
> 
> 江南的雨已经下了整整三天，青石板铺成的街道泛着一层水光，远处的屋檐连成一片墨色的轮廓。
> 
> 范建在书房里已经坐了两个时辰。
> 
> 书桌上摊着一封信，信封上没有署名，只有一个用红笔圈出来的"急"字，墨迹有些晕开，像是送信的人走得太急，或者送信的时候刚好也在下雨。
> 
> 他已经把这封信看了三遍。
> 
> 每一遍的结论都一样——宫里有人要动手了。
> 
> 书房的门被轻轻推开，管家老吴的声音低低地响起："老爷。"
> 
> 范建没有抬头，目光还停在信纸上，手指无意识地在"急"字上摩挲了一下。
> 
> "说。"
> 
> "门外有人，说是宫里来的，要见您。"
> 
> 范建终于抬起头。
> 
> 他的脸色很平静，甚至可以说太平静了，平静得像是根本没有看那封信。
> 
> "让他进来。"
> 
> 老吴躬身退了出去。
> 
> 范建重新看向那封信，沉默了几秒，然后将信纸折好，塞进信封，再放进书桌最下层的抽屉里——那个抽屉里还放着另外三封同样没有署名的信。
> 
> 今天来的，是第四封。

- 平均句长从6字提升到35字
- 长句率从0%提升到53%（大量30-60字的句子）
- 加入时间/地点信息（"庆国十八年，春。"）
- 加入感官描写（青石板、水光、墨色轮廓、晕开的墨迹）
- 对话嵌入叙述中，不是每句对话都单独成段

## 通用句式模板库（直接可用）

### 模板1：动作+描写（25-35字，动作类）
```
[主语] [具体动作]，[细微观察/细节描写]。

例：他伸手拿起茶杯，指尖触到冰凉的瓷面。
例：少年握紧了剑柄，指节因为用力而发白。
例：范建抬起头，目光在信纸上方悬了几秒才落下。
```

### 模板2：描写+叙述推进（30-40字，描写→说明）
```
[场景描写]，[引出新信息/推进情节]。

例：空气里弥漫着药草和铁锈的味道，他想起三个月前也是在这里醒来。
例：窗外的天色已经暗了下来，墙上的挂钟指向凌晨三点。
例：青石板泛着水光，远处的屋檐连成一片墨色——江南的雨，下了三天。
```

### 模板3：长句铺垫 + 极短句揭示（节奏冲击）
```
[30-50字长句，铺垫描写或叙述]
[2-10字极短句，揭示核心信息]

例：他的瞳孔骤然收缩——窗外不是什么都没有，而是有什么东西，贴在玻璃上。
    一张脸。

例：墨水在信纸上晕开了一圈，像是谁在上面滴了一滴水，又像是血。
    第四封。
```

### 模板4：说明/背景推进（20-30字，不带"他感到"）
```
[客观陈述信息]，[解释/推理/后果]。

例：这是斗气大陆的常识——实力说话，弱者没有发言权。
例：青云宗在三千里外，走路要三个月，御剑只需要三天。
例：宫里的人来了三次，第四次，终于不再送心了。
```

### 模板5：对话嵌入叙述（不是"他说：xxx"的格式）
```
[动作/描写]，[对话]

例：老人将瓷碗放在桌上，目光在少年脸上停留了几秒——"青云宗。"
例：管家躬身退到门边，声音压得很低："门外有人，说是宫里来的。"
例：他把信折好塞进抽屉，那个抽屉里还放着另外三封同样的信。
```

## 最终自检清单（写任何章节必查）

**写后5分钟检查**：

1. 随机取100句，计算平均句长 → 目标25字以上
2. 随机取100句，计算30字+长句占比 → 目标40%以上
3. 10字以内短句占比 → 目标20%以下
4. 问号占比 → 悬疑/推理15%+，其他5%-15%
5. 叹号占比 → 军事/仙侠10%+，悬疑/规则怪谈5%以下
6. 最长段落/最短段落比例 → 至少有5:1的变化幅度
7. 检查"他感到/他暗道/他心想" → 全部删掉，直接写内心独白内容
8. 检查"缓缓地/淡淡地/微微地" → 1000字内不超过2个

**写作中实时检查**：
- 写完一段问自己：这段是在**描写**还是在**推进**？如果全是描写，下一段必须推进
- 写完一句问自己：这句话能不能再**加5-10个字**的细节？（让平均句长达标）
- 每写10句问自己：最近10句里有没有至少1个**2-10字的极短句**做节奏点？
