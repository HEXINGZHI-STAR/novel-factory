# 深度去AI味

## 目标

W3 的正则检测只能发现"他感到""突然""缓缓地"这类表层AI味。
本 skill 处理深层AI味——句式均质化、情绪线性化、段落三明治化。

## 检测维度

### 1. 句式均质化
```
症状: 连续5句以上都是20-30字的中等句
修复: 长句(40+)→短句(5-10)→长句交替，制造节奏冲击
```

### 2. 情绪线性化
```
症状: 紧张→更紧张→最紧张（AI的标准爬坡）
修复: 紧张→突然一个日常细节打断→更紧张（真人写作的迂回）
```

### 3. 段落三明治化
```
症状: 每段都是"观点→展开→总结"结构
修复: 允许"只有观点无展开""只有结论无铺垫"的残缺段
```

### 4. 对话均声化
```
症状: 所有角色说话方式一样
修复: 给每个角色一个"口头禅"或"句式偏好"
```

## 执行流程

### Step 1：加载章节

```bash
cat {project}/正文/第{N}章*.txt
```

### Step 2：逐维检测

Claude 对每个维度给出：(1)AI味评分0-1 (2)具体段落引用 (3)改写版本。

### Step 3：生成改写版

Claude 输出完整的改写后正文，标注改动位置和理由。

```markdown
# 改写报告

## 改动1：打破句式均质化（第3-7段）
- 原文: 5句连续22-28字中等句
- 改写: 插入一句3字短句 + 一句45字长描写
- 效果: CV从0.28提升到0.52

## 改动2：打断线性情绪（第8-10段）
- 原文: 紧张→更紧张→最紧张
- 改写: 插入一段日常细节（他注意到茶杯里的茶叶渣）
...
```
