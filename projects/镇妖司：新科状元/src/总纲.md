# 《镇妖司：新科状元》完整大纲

---

## 一、书名

★《镇妖司：新科状元》

备选：《状元斩妖录》《大燕灭妖司》

---

## 二、作品简介

### 一句话（30字）

新科状元被贬镇妖司，满朝以为他活不过三天——没人知道他眼中能看到妖的弱点。

### 完整版（200字）

连中三元的状元沈夜，骑马游街的当天，接到一道密旨：即日起调任镇妖司，任灭妖吏，正九品。

从翰林院的锦绣前程，一脚踩进斩妖除魔的深渊。

他的上司苏青棠断了左臂，脸上三道妖爪疤痕，见他的第一句话是："新来的，你活不过三天。"他的同僚全是残废——瘸腿、瞎眼、哑巴，却每个人都藏着不为人知的秘密。

沈夜有一个能力：他能"看见"妖的弱点。妖物身上会出现红色标记，就像他读圣贤书时标注的重点。他的武器不是刀剑，是《论语》《孟子》《中庸》——圣贤书中的每一个字，都是斩妖的利器。

第一个任务：调查兵部尚书府灭门案。凶手是妖——但府中的妖核黑市名单上，赫然写着他自己的名字。

朝堂之上，有人在用妖核豢养百官。而他体内的那只眼睛，正在一点一点睁开。

### 核心卖点

1. **状元斩妖**：文弱书生 vs 血腥镇妖司。他以圣贤书的逻辑拆解妖物弱点，以朝堂谋略瓦解幕后黑手。从不硬刚，永远先推理再精准击杀。
2. **金手指即诅咒**：能看见妖的弱点——但这只眼睛属于百年前被分尸的天级大妖"观世妖"。用得越多，越接近被夺舍。
3. **全员残废天团**：断臂司主苏青棠、瞎眼神箭阿七、哑巴刀客小陈、瘸腿军师老刘——每个残废都是假象，他们体内的妖核比表面强三个等级，是先帝暗中培养的半妖死士。
4. **圣贤书即斩妖法**：沈夜以《论语》"君子坦荡荡"克敌，以《孟子》"浩然之气"镇妖，以《礼记》"视于无形"破幻——千年前的圣贤早就在书中留下了斩妖秘法。
5. **朝堂皆妖巢**：妖魔只是明面上的敌人，真正的恶藏在奏折里、圣旨里、那些"忠心耿耿"的大臣的血管里。长安城，建在一具上古妖神的尸骨之上。

## 三、人物设定

### 主角：沈夜

- **身份**：连中三元的新科状元，22岁，从翰林院被贬到镇妖司
- **性格**：表面温润如玉，骨子里疯。越冷静越危险。嘴贫但不废话
- **核心信条**："读书不是为了考状元，是为了知道怎么杀你。"
- **金手指**：右眼能看见妖的弱点——红色标记出现在妖物身上，如同圣贤书上的批注
- **金手指真相**：这只眼睛属于天级大妖"观世妖"。二十年前，沈夜是从观世妖尸体里被剖出来的婴儿。他不是完全的人类——他就是观世妖的一部分
- **成长弧线**：从"为什么是我"到"只能是我"再到"幸好是我"

### 女主：苏青棠

- **身份**：镇妖司左司命，32岁，断左臂，脸上三道妖爪疤痕
- **背景**：十年前围剿观世妖，丈夫被妖附身，她亲手杀了他。自己的左臂在那一战被妖吞噬，断口至今有妖气渗血
- **性格**：冷得像刀，狠得像妖。但冷酷底下全是伤。她的封印术印在断臂里，每次使用折寿十年
- **作用**：沈夜的引路人、护身符。第20章以命替沈夜封印妖眼。第75章被沈夜以帝级本源救回

### 同僚天团

- **瘸腿老刘**：表面瘸腿，实则腿里嵌着半截王级妖骨。年轻时被妖骨寄生，靠圣贤书的正气压制。镇妖司的活字典
- **瞎眼阿七**：表面失明，实则耳朵能听见十里内妖物的心跳。他的眼睛不是瞎的，是被自己挖掉的——为了不给观世妖留后门
- **哑巴小陈**：表面哑巴，实则会腹语。后颈有一排牙印，体内寄宿着一只幼年妖虫。他是先帝放在镇妖司的最后一道保险

### 核心反派

- **魏忠**：司礼监掌印太监，妖核黑市的真正主人。体内寄生着观世妖的尾巴——他养了沈夜二十年，等着收割那只右眼
- **皇帝燕无极**：看似昏庸，实则体内妖气被龙气压制。他在下一盘大棋——让沈夜入镇妖司是他故意布的局。不是反派，是被困在龙椅上的棋手
- **观世妖**：被分尸封印的天级大妖。右眼在沈夜体内，左眼在首辅手中，尾巴在魏忠体内。一旦三件归一，可灭世

## 写作建议

1. **句均25-30字**，对话率35-40%，参考《龙符》节奏
2. **每章末必须留钩子**，类型按上述分布轮流
3. **沈夜从不硬刚**，永远先用圣贤书分析→找到弱点→精准击杀
4. **同僚的残废都是伏笔**，每10章左右揭露一个秘密
5. **苏青棠的情感线**在第20章达到高潮，后续作为沈夜的精神支柱# 《镇妖司：新科状元》第二卷：破局（第26-50章）

26. 暗棋初现 → 沈夜在兵部尚书府废墟中发现妖核黑市交易暗账，上面赫然有镇妖司内部代号 → 钩子：暗账最后一页写着“苏青棠，三日前交易”

27. 同僚异动 → 瘸腿老刘深夜独自离司，沈夜跟踪发现他进入内阁首辅府邸后门 → 钩子：老刘转身时，瘸腿的左脚踩碎了青石板

28. 瞎眼辨妖 → 阿七“盲眼”突然精准指出沈夜袖中暗账的墨迹年份，说这是三年前的旧账 → 钩子：阿七说完，七窍同时流血，倒地抽搐

29. 哑巴开口 → 小陈在阿七昏迷后，突然用腹语对沈夜说：“他们不是残废，是故意被废的” → 钩子：小陈说完，脖子上的喉结开始蠕动，像有什么东西要钻出来

30. 首章伏笔回收 → 沈夜发现传密旨太监的影子在月光下没有轮廓——那是王级妖物“影妖”的标记 → 钩子：影妖从太监身体里剥离，扑向沈夜

**【小高潮】第30章：沈夜用圣贤书“格物致知”之理，发现影妖弱点在“光源折射角度”，以铜镜阵困杀影妖，救下太监**

31. 太监之口 → 被救太监交代：密旨是内阁首辅伪造，目的是把沈夜送入镇妖司当“饵料” → 钩子：太监说完，七窍流出黑色脓血，化作一具空壳

32. 六部密会 → 沈夜潜入吏部，发现六部尚书每夜子时在密室开会，桌上摆着妖核磨成的粉末 → 钩子：密室墙上挂着苏青棠十年前未断臂的画像

33. 断臂真相（上） → 苏青棠醉酒后露出断臂，伤口处有妖气缠绕，她说：“这只手是被自己人砍的” → 钩子：她袖中滑落一块令牌，上面刻着“镇妖司·暗部·甲等”

34. 老刘的腿 → 沈夜用《黄帝内经》中“筋脉逆行”理论，推断老刘的瘸腿是长期佩戴某种重物所致 → 钩子：老刘掀开裤腿，小腿上绑着三十斤的玄铁锁链，锁链上刻满封印咒文

35. 阿七的眼 → 阿七醒来，说自己的眼睛被挖掉前，看到过“皇帝跪在地上，向首辅磕头”的场景 → 钩子：沈夜追问时间，阿七说：“那是三年前，先帝驾崩当晚”

**【小高潮】第35章：沈夜通过时间线推理，发现皇帝登基前夜，内阁首辅曾以“护国”之名，用妖术篡改皇位继承**

36. 小陈的喉 → 小陈用腹语告诉沈夜：他的舌头被割掉前，曾经唱过一首“镇妖司的挽歌”，歌词是暗号 → 钩子：沈夜破解歌词，发现是镇妖司地下密室的开启咒文
