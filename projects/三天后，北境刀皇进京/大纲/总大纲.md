# 一、书名（3个候选，选最好的标★）

1. 《三天后，北境刀皇进京》★
2. 《御前三日：刀镇京城》
3. 《谢寒山：我只需三天》

# 二、作品简介

## 一句话版（30字以内）
北境第一刀客进京第一天，被嫁祸杀人，只有三天找出真凶。

## 完整版（200字左右）
谢寒山，北境边军第一刀客，十年杀敌三千，刀下从无活口。因战功调入京城任御前四品带刀侍卫，进京第一天就摊上大事——太和殿屋顶出现一具尸体，死者是北燕使臣，死因是谢家刀法。

有人要让他死。

主和派要用他的人头向北燕求和，主战派要借他的人头挑起战争。满朝文武，没有一个想让他活。可他们不知道，谢寒山在战场上活下来，靠的不是运气。

他只有三天。三天后，北燕十万大军压境。朝堂上，有人笑着敬他毒酒，有人跪着求他赴死。谢寒山把刀往桌上一拍：“想让我死？先问问它答不答应。”

当所有人都以为他只是个莽夫时，谢寒山已经拔刀——不是杀人，是斩开一个局。

# 三、核心卖点（5条）

1. **开篇即高潮**：主角进京第一天就被嫁祸，三天倒计时逼命，节奏快到喘不过气
2. **刀刀见血不废话**：主角杀伐果断，遇到威胁直接拔刀，没有圣母心、不内耗、不解释
3. **朝堂比战场更狠**：表面是武侠，内核是权谋，敌人笑着敬酒比战场上的刀更致命
4. **每章都有打脸**：不管是文臣还是武将，敢动谢寒山的人，下章必被当场打脸
5. **情绪拉满不拖沓**：每一章的核心是“读者爽了没”，不是“剧情走没走”，打脸、逆袭、反杀轮番上

# 四、人物设定

## 主角：谢寒山
- **身份**：北境边军第一刀客→御前四品带刀侍卫
- **性格**：话少、手狠、不解释、不废话。战场上养成的习惯——能动手绝不动口
- **核心矛盾**：他习惯的是明刀明枪的战场，但京城是笑里藏刀的暗局。他必须在三天内学会“用脑子杀人”
- **作用**：爽点发动机，每章至少打一次脸或杀一个人

## 女主：沈清漪
- **身份**：大理寺少卿之女，京城第一女捕头
- **性格**：冷静、聪明、嘴硬心软。看不上谢寒山这种“粗人”，但不得不跟他合作
- **核心矛盾**：她相信证据和律法，谢寒山只信刀。两人办案方式冲突不断
- **作用**：提供破案线索、制造情感冲突、关键时刻被谢寒山救下产生情愫

## 关键配角：赵无忌
- **身份**：谢寒山在边军的老战友，现为京畿卫指挥使
- **性格**：表面粗犷豪爽，实则心思缜密
- **核心矛盾**：他想帮谢寒山，但自己的位置不允许他明着站队
- **作用**：暗中提供情报、关键时刻出手相助、后期揭露他才是幕后黑手之一（反转）

## 反派：宰相李崇文
- **身份**：当朝宰相，主和派领袖
- **性格**：笑面虎，说话永远温文尔雅，杀人永远借刀
- **核心矛盾**：他需要谢寒山的人头来平息北燕怒火，保住自己的相位
- **作用**：明面上的最大敌人，让主角吃尽苦头，最终被主角当朝打脸

## 隐藏反派：三皇子萧景琰
- **身份**：皇帝第三子，主战派幕后推手
- **性格**：表面谦和，实则野心勃勃
- **核心矛盾**：他想借谢寒山的刀挑起战争，趁机夺权
- **作用**：真正的幕后黑手，结局反转的爆点

# 五、世界观要点（5条）

1. **大梁王朝**：架空古代王朝，北境常年与北燕交战，朝廷分主战、主和两派
2. **刀客体系**：以刀法论高低，谢家刀法为北境第一，但谢寒山是唯一传人
3. **朝堂等级**：御前侍卫分九品，四品可带刀上殿，但无权参与朝政
4. **三天倒计时**：北燕使臣被杀，北燕要求三日内交出凶手，否则十万大军南下
5. **京城暗网**：表面是天子脚下，实则各方势力盘根错节，连皇宫都不安全

# 六、大纲（按番茄风格，共60章，每章约2000字）

## 第一卷：刀入京城（1-15章）

**第1章：进京第一天，有人要杀我**
- 冲突开场：谢寒山刚进城门，一队禁军围上来，说太和殿屋顶有尸体
- 死者是北燕使臣，伤口是谢家刀法
- 谢寒山看了一眼尸体：“这刀法不是我用的，力道不对”
- 没人信他，主和派当场要抓人
- 谢寒山拔刀：“想抓我？试试”
- 情绪点：憋屈→愤怒→拔刀震慑全场

**第2章：笑着敬酒的人最危险**
- 宰相李崇文设宴“接风”，满朝文武都在
- 李崇文笑着敬酒：“谢将军，喝了这杯，万事好商量”
- 谢寒山闻出酒里有毒，把酒杯摔在地上
- 李崇文脸色不变：“谢将军这是不给本官面子？”
- 谢寒山：“你配吗？”
- 情绪点：压抑→爆发→爽

**第3章：三天倒计时**
- 北燕使者送来国书：三天内不交出凶手，十万大军南下
- 皇帝召见谢寒山，给他三天时间自证清白
- 谢寒山走出大殿，沈清漪拦住他：“我帮你查案”
- 谢寒山：“不需要”
- 沈清漪：“你一个人能在京城活三天？”
- 情绪点：孤立无援→有人伸出援手（但主角不领情）

**第4章：第一个线索**
- 谢寒山去案发现场，发现死者指甲里有黑色粉末
- 沈清漪认出是“西域断肠草”，京城只有三家药铺有
- 两人分头查药铺，谢寒山第一家就遇到埋伏
- 十个杀手从暗处杀出，谢寒山一刀斩五个
- 情绪点：紧张→反杀→爽

**第5章：药铺老板死了**
- 谢寒山赶到第二家药铺，老板已经被人灭口
- 沈清漪赶到，发现老板死前留下一个字：“三”
- 两人推测是“三皇子”或“三日后”
- 赵无忌出现，说京畿卫查到一批西域毒药流入三皇子府
- 情绪点：线索断了→又接上→悬念加深

**第6章：三皇子请客**
- 三皇子萧景琰设宴，请谢寒山赴约
- 宴会上，三皇子暗示自己能帮他洗脱罪名
- 条件是：谢寒山要公开指认宰相李崇文是幕后黑手
- 谢寒山：“你们一个要我死，一个要我当枪，当我傻？”
- 三皇子脸色一变：“谢将军，你只有三天”
- 情绪点：被算计→看穿阴谋→反将一军

**第7章：夜闯宰相府**
- 谢寒山决定主动出击，夜闯宰相府找证据
- 潜入书房，发现李崇文与北燕使者的密信
- 正要拿走，李崇文带人出现：“谢将军，深夜来访，所为何事？”
- 谢寒山：“来拿你的命”
- 情绪点：紧张→对峙→拔刀

**第8章：一刀破局**
- 谢寒山一刀劈开书桌，密信被震飞
- 赵无忌带京畿卫“恰好”赶到，捡起密信
- 李崇文脸色铁青：“你们串通好了？”
- 谢寒山：“你教我的，京城办事，要讲证据”
- 情绪点：被陷害→反击成功→爽

**第9章：密信是假的**
- 密信被送到大理寺鉴定，是伪造的
- 沈清漪发现纸张是三个月前才产的，而密信日期是半年前
- 谢寒山意识到：有人设了一个更大的局
- 真正的凶手不是李崇文，也不是三皇子
- 情绪点：刚看到希望→再次陷入绝望

**第10章：赵无忌的背叛**
- 谢寒山去找赵无忌，发现他在密室跟一个黑衣人见面
- 偷听到对话：“谢寒山必须死，他知道太多了”
- 谢寒山冲进去，赵无忌拔刀相对
- 赵无忌：“兄弟，对不住了，我也是身不由己”
- 情绪点：信任崩塌→愤怒→心寒

**第11章：沈清漪的发现**
- 沈清漪重新验尸，发现死者真正的致命伤不是刀伤
- 是毒，刀伤是死后补的
- 她查到这种毒只有皇宫才有——是御用的“鹤顶红”
- 凶手在皇宫里
- 情绪点：柳暗花明→新的方向

**第12章：皇宫里的影子**
- 谢寒山以御前侍卫身份进宫，调查御药房
- 发现少了一瓶鹤顶红，登记册上写着“三皇子府取用”
- 但三皇子府的管事说没收到过
- 有人冒领了毒药
- 情绪点：接近真相→又有迷雾

**第13章：最后一天**
- 第二天就是北燕大军压境的日子，谢寒山只剩一天
- 皇帝下旨：明天午时，若查不出真凶，谢寒山斩首示众
- 沈清漪急了：“你到底有没有把握？”
- 谢寒山：“我从来不做没把握的事”
- 情绪点：倒计时压迫感→主角的自信

**第14章：真凶浮出水面**
- 谢寒山把所有线索串起来：假密信、灭口药铺老板、御用鹤顶红
- 所有线索都指向一个人——皇帝身边的太监总管，曹正淳
- 曹正淳是北燕安插在大梁的卧底
- 他杀使臣，嫁祸谢寒山，目的是挑起大梁内乱
- 情绪点：真相大白→愤怒→准备收网

**第15章：殿前对决**
- 第二天午时，谢寒山被押上太和殿
- 皇帝问：“谢寒山，你还有什么话说？”
- 谢寒山：“有，真凶就在这大殿上”
- 他看向曹正淳：“曹公公，你说是吗？”
- 曹正淳脸色不变：“谢将军，你血口喷人”
- 谢寒山把证据一件件摆出来，曹正淳脸色越来越白
- 最后，谢寒山拔刀：“曹公公，接我一刀”
- 一刀斩下，曹正淳的假发掉落，露出北燕人的秃顶
- 全场哗然
- 情绪点：压抑→爆发→爽到顶点

## 第二卷：京城暗流（16-30章）

**第16章：皇帝的秘密任务**
- 真相大白后，皇帝单独召见谢寒山
- 原来皇帝早就知道曹正淳是卧底，故意设局引蛇出洞
- 谢寒山只是棋子，但皇帝欣赏他的能力
- 皇帝交给他一个新任务：查清朝中还有多少北燕卧底
- 情绪点：被利用→得到信任→新的挑战

**第17章：新的敌人**
- 谢寒山刚出宫，就遇到刺杀
- 这次来的不是普通杀手，是北燕的“影卫”
- 影卫首领说：“谢寒山，你坏了我们的大事，必须死”
- 谢寒山：“想杀我的人多了，你算老几？”
- 情绪点：危机→反击→霸气

**第18章：沈清漪的告白**
- 沈清漪帮谢寒山包扎伤口，两人关系升温
- 沈清漪：“你为什么总是一个人扛？”
- 谢寒山：“习惯了，战场上没人能靠”
- 沈清漪：“现在有了”
- 两人对视，气氛暧昧
- 情绪点：温情→心动→打破心防

**第19章：暗网初现**
- 谢寒山查到一个叫“暗网”的组织，专门帮北燕传递情报
- 暗网成员遍布朝野，连皇亲国戚都有
- 他需要找到一个突破口
- 赵无忌主动找上门：“我可以帮你，但你要答应我一件事”
- 情绪点：新的线索→信任危机→交易

**第20章：赵无忌的条件**
- 赵无忌的条件：事成之后，谢寒山要帮他除掉三皇子
- 原来三皇子知道赵无忌贪污军饷的证据，一直在威胁他
- 谢寒山：“你为了自保，就要我杀人？”
- 赵无忌：“在这京城，不是你死就是我活”
- 情绪点：道德困境→妥协→无奈

**第21章：暗网的接头人**
- 根据赵无忌的情报，暗网在京城有个接头人——醉仙楼的老板娘
- 谢寒山和沈清漪假扮成酒客去调查
- 老板娘一眼看穿他们的身份：“两位，不是来喝酒的吧？”
- 谢寒山：“我是来要你命的”
- 情绪点：紧张→对峙→冲突

**第22章：老板娘的秘密**
- 老板娘叫苏媚，曾经是北燕的公主
- 她被迫嫁到大梁和亲，丈夫死后被北燕利用
- 苏媚：“我可以帮你们，但你们要帮我离开京城”
- 谢寒山：“成交”
- 情绪点：反转→同情→合作

**第23章：暗网的名单**
- 苏媚拿出一份名单，上面是暗网在大梁的所有成员
- 名单上有三皇子、李崇文、还有几个武将
- 沈清漪震惊：“这么多人？”
- 谢寒山：“难怪北燕每次打仗都知道我们的部署”
- 情绪点：震撼→愤怒→决心

**第24章：第一个目标**
- 谢寒山决定先动名单上的武将——镇北将军王霸
- 王霸掌握北境十万大军，如果是卧底，后果不堪设想
- 谢寒山直接带刀上门：“王将军，聊聊？”
- 王霸心虚，下令士兵抓人
- 谢寒山一刀劈开中军帐：“我看谁敢动？”
- 情绪点：霸气→压制→爽

**第25章：王霸的供词**
- 王霸招供，他是被三皇子拉下水的
- 三皇子许诺他，事成之后让他当兵马大元帅
- 谢寒山把供词呈给皇帝，皇帝震怒
- 但三皇子是皇亲，皇帝不能动他
- 情绪点：真相→无奈→憋屈

**第26章：皇帝的态度**
- 皇帝告诉谢寒山：“朕知道三皇子有野心，但他毕竟是朕的儿子”
- “你查案可以，但不能动他”
- 谢寒山：“那如果他造反呢？”
- 皇帝沉默了很久：“那就按律法办”
- 情绪点：得到默许→新的目标→暗流涌动

**第27章：三皇子的反击**
- 三皇子知道谢寒山在查他，开始反击
- 他派人绑架了沈清漪，要挟谢寒山停手
- 谢寒山收到信：“明天午时，城外破庙，一个人来”
- 否则，就等着收尸
- 情绪点：危机→愤怒→不顾一切

**第28章：单刀赴会**
- 谢寒山一个人来到破庙，三皇子带着一百个刀手等着他
- 三皇子：“谢寒山，你太狂了，真以为我拿你没办法？”
- 谢寒山：“我狂，是因为我有狂的资本”
- 拔刀，一人对百人
- 情绪点：绝望→热血→燃

**第29章：血战破庙**
- 谢寒山以一敌百，刀刀见血
- 三皇子没想到他这么能打，下令放箭
- 谢寒山抓起一个刀手当盾牌，冲到三皇子面前
- 刀架在三皇子脖子上：“放人”
- 三皇子吓尿了：“放、放人”
- 情绪点：压抑→爆发→爽到爆

**第30章：沈清漪的转变**
- 沈清漪被救出来，看到满地的尸体，对谢寒山刮目相看
- 她以前觉得谢寒山是个莽夫，现在发现他比任何人都聪明
- 沈清漪：“你为了我，值得吗？”
- 谢寒山：“我说过，现在有人可以靠了”
- 两人相拥，感情升温
- 情绪点：感动→甜蜜→关系确定

## 第三卷：刀指苍穹（31-45章）

**第31章：三皇子的底牌**
- 三皇子被谢寒山威胁后，彻底撕破脸
- 他联合李崇文，准备发动政变
- 他们的计划：在皇帝寿宴上动手，控制皇宫，逼皇帝退位
- 谢寒山从
