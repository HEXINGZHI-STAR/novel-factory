# 篮神崛起 总大纲

## 一句话核心概念
普通大学生林风意外获得篮球神级系统，能复制任何人的技能，从此从校队饮水机管理员一路逆袭，横扫CUBA、CBA、NBA，成为一代篮神！

## 核心卖点
1. 系统流，技能复制，开局暴爽
2. 七猫风格，节奏快，每章有爽点
3. 情感线丰富，友情爱情亲情
4. 升级体系清晰，系统奖励丰厚

## 主要人物
### 主角：林风
- 身份：普通大学生，校队饮水机管理员
- 性格：坚韧、重情义、有恩报恩有仇报仇
- 能力：复制系统，可以复制对手或队友的任何技能
- 核心目标：成为NBA巨星，带领中国队拿到奥运冠军

### 女主角：苏雅
- 身份：校队经理，校花学霸
- 性格：外冷内热，温柔善良
- 能力：篮球战术天才
- 秘密：爷爷是前国手

### 死党：胖子
- 身份：林风发小，校队替补中锋
- 性格：幽默搞笑，重义气
- 作用：主角的左膀右臂

## 章节规划（每章约2000字）
1. 第1章：觉醒！神级复制系统
2. 第2章：首秀！把校队队长打爆
3. 第3章：震惊！一战成名
4. 第4章：校花的特别关注
5. 第5章：赌约！要打CUBA预选赛
